# Misskey Nord Theme
Theme for misskey based on Nord theme color

# Use

Paste this on misskey.

```
{
	id: 'Nord',

	name: 'Nord',
	author: 'KazukyAkayashi',
	desc: 'Theme based on Nord theme color',

	base: 'dark',

	props: {
	accent: '#8fbcbb',
	tag:'#a3be8c',
	bg: '#2e3440',
	fg: '#eceff4',
	panel: '#3b4252',
	header: 'rgba(46, 52, 64, 0.75)',
	renote: '#17bf63',
	link: '@accent',
	mention: '@accent',
	hashtag: '@tag',
	},
}
```
