# Misskey Nord Theme
Theme for misskey based on [Nord theme color](https://www.nordtheme.com/#palettes-modularity).

# Use

Copy this code on misskey theme editor :

```
{
	id: 'Nord',

	name: 'Nord',
	author: 'KazukyAkayashi',
	desc: 'Theme based on Nord theme color',

	base: 'dark',

	props: {
	accent: '#8fbcbb',
	tag:'#a3be8c',
	bg: '#2e3440',
	fg: '#eceff4',
	fgHighlighted: '@link',
	panel: '#3b4252',
	navFg: '@fg',
	navHoverFg: '@accent',
	navActive: '@accent',
	navIndicator: '@accent',
	header: 'rgba(59, 66, 82, 0.75)',
	link: '#b48ead',
	mention: '#bf616a',
	hashtag: '@tag',
	renote: '#d08770',
	pageBg: '@bg',
	badge: '#a3be8c',
	divider: 'rgba(46, 52, 64, 0.7)',
	driveFolderBg: '@panel',
	},
}
```
